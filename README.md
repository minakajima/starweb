# 🌌 天の川撮影プランナー

天体写真愛好家のための撮影計画支援Webアプリです。指定した撮影場所と日時から天の川の見え方をシミュレーションし、最適な撮影日時を自動で算出します。

## 機能

- **最適日時を探す** — 撮影場所を入力すると、今後12ヶ月間で銀河中心が最も高く昇る日時をTOP N件表示します
- **日時指定シミュレーション** — 任意の日時・場所で天の川の見え方を写真風に描画します
- **天気予報** — 7日以内の撮影日はOpen-Meteoの雲量データを取得して快晴度（★1〜5）を表示します
- **画角オーバーレイ** — カメラの焦点距離・センサーサイズを入力すると、実際の撮影画角を空の図に重ねて表示します
- **お気に入り登録** — よく使う撮影場所を保存し、すぐに呼び出せます
- **設定** — カメラのデフォルト値や検索条件をブラウザに記憶します

## スクリーンショット

| 最適日時検索 | 空シミュレーション |
|---|---|
| 場所を入力してTOP 5件を表示 | スライダーで日時をリアルタイム調整 |

## 使い方

### 最適日時を探す

1. 📍 撮影場所に地名を入力して「検索」を押す（または地図をクリック）
2. 「✨ 計算する」を押す
3. 表示された候補カードをクリックすると、その日時の空の図と撮影条件が表示される

### 日時指定シミュレーション

1. 「🔭 日時指定シミュレーション」タブを選択
2. 撮影場所を選択する
3. 日付・時刻スライダーで日時を調整 → 空の図がリアルタイムで更新される

### 画角枠を表示する

1. 「📷 カメラ設定」に焦点距離（mm）とセンサーサイズを入力
2. カメラ向き（縦/横）を選択
3. 「画角枠を表示」トグルをONにする

### お気に入りに追加する

1. 撮影場所を選択した後、☆ボタンをクリック
2. ⭐ お気に入りボタンから保存した場所を呼び出せる
3. ✏️ で名前の変更、🗑️ で削除が可能

## 技術スタック

| 用途 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（バニラ、ビルドツール不要） |
| 地図 | [Leaflet.js](https://leafletjs.com/) 1.9.4 + OpenStreetMap |
| ジオコーディング | [Nominatim](https://nominatim.openstreetmap.org/) API |
| 薄明・月齢計算 | [SunCalc](https://github.com/mourner/suncalc) 1.8.0 |
| 天気予報 | [Open-Meteo](https://open-meteo.com/) API（無料・APIキー不要） |
| 天文計算 | オリジナル実装（JD / GMST / 赤道座標↔地平座標 / 銀河座標変換） |
| 空の描画 | Canvas 2D API（グノモニック投影） |
| データ永続化 | localStorage |

## セットアップ

ビルド不要です。ファイルをWebサーバーに配置するだけで動作します。

```bash
# ローカルで確認する場合（Python）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

または `index.html` をブラウザで直接開いても動作します（Nominatimへのリクエストに `file://` 制限がかかる場合はローカルサーバー推奨）。
