# 天の川撮影プランナー — ソフトウェア仕様書

## 1. 概要

**天の川撮影プランナー**は、天体写真愛好家が天の川の撮影計画を立てるためのシングルページWebアプリケーションです。撮影場所と日時から天の川（銀河中心）の空における位置・見え方をシミュレーションし、最適な撮影条件を提示します。

### 主な機能

| 機能 | 説明 |
|------|------|
| 最適日時検索 | 指定場所で今後12ヶ月間の最適撮影日時TOP N件を算出 |
| 空シミュレーション | 任意の日時・場所で天の川の見え方を写真風に描画 |
| 天気予報表示 | Open-Meteo APIによる雲量・快晴度評価（7日以内） |
| 画角オーバーレイ | カメラ焦点距離・センサーサイズから画角枠を表示 |
| お気に入り管理 | よく使う撮影場所をlocalStorageへ保存・管理 |
| 設定画面 | デフォルト値をlocalStorageへ永続保存 |

## 4. 天文計算仕様

### 4.1 座標変換

#### ユリウス日（JD）

```
JD = Unix時刻(ms) / 86400000 + 2440587.5
```

#### グリニッジ恒星時（GMST）

IAU 1982 式：

```
T = (JD - 2451545.0) / 36525
GMST(°) = 280.46061837 + 360.98564736629 × (JD - 2451545.0)
         + 0.000387933 × T² - T³ / 38710000
```

#### 地方恒星時（LST）

```
LST = (GMST + 経度) mod 360
```

#### 赤道座標 → 地平座標（Alt/Az）

時角 `HA = (LST - RA) mod 360`

```
sin(Alt) = sin(Dec)×sin(Lat) + cos(Dec)×cos(Lat)×cos(HA)
Az = atan2(-cos(Dec)×sin(HA),  cos(Lat)×sin(Dec) - sin(Lat)×cos(Dec)×cos(HA))
```

#### 銀河座標 → 赤道座標（J2000）

IAU銀河極座標：
- NGP赤経 `α_NGP = 192.85948°`
- NGP赤緯 `δ_NGP = 27.12825°`
- 昇交点の銀経 `l_asc = 32.93192°`

```
sin(Dec) = sin(δ_NGP)×sin(b) + cos(δ_NGP)×cos(b)×cos(l - l_asc)
RA = α_NGP + atan2(cos(b)×sin(l - l_asc),
                   cos(δ_NGP)×sin(b) - sin(δ_NGP)×cos(b)×cos(l - l_asc))
```

### 4.2 銀河中心の座標

| 属性 | 値 |
|------|-----|
| 銀経 l | 0° |
| 銀緯 b | 0° |
| 赤経 RA (J2000) | 266.417° |
| 赤緯 Dec (J2000) | −29.008° |

### 4.3 最適日時オプティマイザ

1. 本日から365日間を1日単位でスキャン
2. SunCalc `times.night`（天文薄明終了）〜翌日 `times.nightEnd`（天文薄明開始）を取得
3. 15分刻みで銀河中心高度を計算し、闇夜中の最大高度と最適時刻を記録
4. 最小高度（設定値、デフォルト5°）を下回る日は除外
5. スコア計算:
   ```
   altScore  = max(0, maxAlt) / 90
   moonScore = 新月前後(±位相0.15以内)なら輝度から算出、それ以外 = 0
   score = altScore × 0.6 + moonScore × 0.4
   ```
6. スコア降順でソート → 7日以内の重複をクラスタリングで除外
7. 上位N件を返す（N = 設定値）

### 4.4 天気予報

- API: `https://api.open-meteo.com/v1/forecast`
- パラメータ: `hourly=cloud_cover`, `forecast_days=7`, `timezone=auto`
- 雲量（%）→ 快晴度評価の変換:

| 雲量 | 評価 | ★ |
|------|------|---|
| 0〜10% | 絶好 | ★★★★★ |
| 11〜25% | 良好 | ★★★★☆ |
| 26〜50% | 普通 | ★★★☆☆ |
| 51〜75% | 不向き | ★★☆☆☆ |
| 76〜100% | 厳しい | ★☆☆☆☆ |

- 7日以内の日時のみ表示（`isWithin7Days` 判定）
- 同一座標のデータはセッション内にキャッシュ

## 5. 描画仕様

### 5.1 グノモニック投影

カメラ方向（銀河中心の Alt/Az）を中心に、天球をキャンバス（500×340px）へ矩形投影します。

```
fwd = altAzToVec(cAlt, cAz)           // カメラ光軸
up  = 天頂方向をfwdに直交するよう補正
right = up × fwd                       // 右方向

焦点距離f = (VIEW_W/2) / tan(HFOV/2)
x = VIEW_W/2 + f × (v·right) / (v·fwd)
y = VIEW_H/2 - f × (v·up)   / (v·fwd)
```

- 参照ビュー画角（HFOV）: 設定値（デフォルト 110°）
- 銀河中心が地平線5°以上: 銀河中心を向く。それ以外: 南・高度30°

### 5.2 描画レイヤー（奥から順）

| レイヤー | 内容 |
|---------|------|
| 1. 空背景 | 上から #000007 → #060d22 のグラデーション＋エアグロー |
| 2. 地平線・地面 | Alt=0°ラインをトレース、地面を暗緑色で塗りつぶし |
| 3. 地形シルエット | 複数サイン波の重ね合わせによる山稜シルエット |
| 4. 星 | 3000点を疑似乱数（mulberry32）で全天配置。輝度差あり、輝星はグロー付き |
| 5. 天の川帯 | 銀河面（b=0°）を360点サンプリング。中心付近は暖色（橙〜金）、周辺は青白 |
| 6. ダストレーン | b=±2.5°に沿った暗い帯 |
| 7. 銀河バルジ | l=±18°, b=±10°の参照点から楕円の向き・大きさを決定し9レイヤー楕円で描画 |
| 8. 月 | SunCalcによる月位置・月齢で色・グロー変化 |
| 9. 銀河中心マーク | 照準リング＋ラベル |
| 10. 方位ヒント | 中央下部に方位表示（例：「↑ 南南西」） |
| 11. 画角枠 | カメラ設定入力時に白い矩形で表示（オプション） |

### 5.3 画角オーバーレイ

センサーサイズとカメラ向き（縦/横）から画角を計算してキャンバス上に矩形表示します。

```
HFOV_cam = 2 × atan(センサー幅 / (2 × 焦点距離))
VFOV_cam = 2 × atan(センサー高さ / (2 × 焦点距離))
// 縦向きの場合: 幅と高さを入れ替え
```

センサーサイズ定義:

| キー | 幅 (mm) | 高さ (mm) |
|------|---------|----------|
| full | 36.0 | 24.0 |
| apsc | 23.5 | 15.6 |
| m43  | 17.3 | 13.0 |

## 6. データ永続化（localStorage）

### 6.1 設定

キー: `starweb-settings`

```json
{
  "focalLength":  "",
  "sensorKey":    "full",
  "orientation":  "portrait",
  "showFov":      false,
  "viewHFov":     110,
  "optimalCount": 5,
  "minGcAlt":     5
}
```

### 6.2 お気に入り

キー: `starweb-favorites`

```json
[
  {
    "id":       1711600000000,
    "name":     "富士山",
    "fullName": "富士山, 富士吉田市, 山梨県, ...",
    "lat":      35.3606,
    "lon":      138.7274
  }
]
```

- `id`: `Date.now()` で生成するユニークID
- `name`: 表示用の短縮名（カンマ前の部分、編集可能）
- `fullName`: Nominatim が返す完全な地名
- 座標の同一判定: `|Δlat| < 0.001 かつ |Δlon| < 0.001`

## 8. リアルタイム描画

シミュレーションモードのスライダー操作中は `requestAnimationFrame` を使って毎フレーム描画をスケジュールします。

```javascript
let _rafPending = false;
function scheduleSimRender(date) {
  _pendingDate = date;
  if (_rafPending) return;
  _rafPending = true;
  requestAnimationFrame(() => {
    _rafPending = false;
    drawSky(canvas, lat, lon, _pendingDate, ...);
    updateConditionPanel(_pendingDate, ...);
  });
}
```

## 9. エラーハンドリング

| 状況 | 挙動 |
|------|------|
| 場所未選択で計算 | エラーメッセージ表示 |
| Nominatim検索失敗 | 「見つかりませんでした」表示 |
| Open-Meteo取得失敗 | 天気欄を非表示 |
| 白夜・極夜（天文薄明なし） | その日をスキップ |
| 銀河中心が地平線以下 | デフォルト方向（南・30°）で描画 |
| localStorage読み書き失敗 | try/catchで無視（デフォルト値使用） |
